# 🤖 飞书 + 扣子机器人

一个简单的中转服务，让你的扣子AI机器人可以在飞书中使用。

## 快速开始（3步搞定）

### 第1步：填写配置
打开 `main.py`，修改第12-20行：
```python
FEISHU_APP_ID = "你的飞书App_ID"
FEISHU_APP_SECRET = "你的飞书App_Secret"
COZE_API_TOKEN = "你的扣子Token"
```

### 第2步：部署上线
选择一种方式：
- **最简单**：用Vercel（免费，5分钟部署）
- **中国用户**：用腾讯云函数（免费，访问快）
- **有服务器**：直接运行 `python3 main.py`

详细步骤看：`部署指南.md`

### 第3步：配置飞书
1. 在飞书开放平台配置回调地址
2. 添加事件订阅：`im.message.receive_v1`
3. 发布应用版本

## 文件说明
- `main.py` - 主程序（修改这个文件的配置）
- `requirements.txt` - 依赖包列表
- `部署指南.md` - 详细部署教程

## 工作原理
```
用户在飞书发消息 
    ↓
中转服务接收消息
    ↓
调用扣子API
    ↓
返回AI回复到飞书
```

## 注意事项
⚠️ 请妥善保管你的 Token 和 Secret，不要泄露！
⚠️ 第一次使用需要在飞书中添加机器人
